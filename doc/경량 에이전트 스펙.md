# 파이썬 CLI 기반 AI Agent 개발

## 개요
### 목적: AI Agent의 동작 방식 이해

### 기능:
1. 로컬에 저장된 파일 내용에서 특정 단어, 구를 검색/탐색
2. DB Connection을 이용하여 DB에서 필요한 정보를 검색
3. 사용자 의도에 맞는 형식(템플릿)으로 출력

## Agent 구조
1. 사용자 입력
2. LLM Api 요청/응답 처리
3. 내장 Skill/Tool 관리
4. loop/pipeline
5. Tool 실행
6. Tool 결과 관리
7. System Prompt 관리
8. 출력 형식 관리
9. 작업 폴더 지정

## Agent 동작 방식
1. 사용자 질문 입력
2. Claude API를 이용해서 사용자 질문을 분해후 내장 Tool과 매칭하여 Tool 실행 절차를 JSON 형식으로 반환
3. Tool 실행
4. Tool 실행 결과를 구조화된 JSON Context로 변환
5. 사용자 질문의 의도 + JSON Context + 출력 형식을 조합하여 출력용 LLM에게 요청할 프롬프트 작성
6. Claude API를 이용해서 출력용 LLM 요청
7. LLM 응답을 사용자에게 출력
